Primer3Plus¶

Release v0.1.3. (Changelog)

Github |gitpage|

Primer3Plus is a Python DNA primer design tool based off of primer3 1 2 and the Python primer3 wrapper https://pypi.org/project/primer3-py/.

design = Design()
design.set.template("AGCGTCGTGTATGGTAGTGTATTTGCGTTGACGTTGCTGACGTCGTTGAGTCGT")
design.set.as_cloning_task()
design.run()

design = Design()
design.set.template("AGCGTCGTGTATGGTAGTGTATTTGCGTTGACGTTGCTGACGTCGTTGAGTCGT")
design.set.as_cloning_task()
design.set.left_sequence("AGCGTCGTGTATGGTAGTG")
design.run_and_optimize(5)

Installation¶

pip install primer3plus -U
